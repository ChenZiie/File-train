This Project is for 2022spring MSBD5017 based on Etherdata Blockchain. 
The Demo can be viewed at https://file-train-8plqcr1yf-block-chain-file-train-project.vercel.app/

## Setups
Install dependencies

```
yarn
```


Test

```
yarn test
```

Start dev server

```
yarn dev
```
